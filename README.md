## Requirements

pyautogui
aiohttp
asyncio
requests
pygetwindow

## Run

```
pip install -r requirements.txt
```

```
python main.py
```

## Capability

1. Interactive coordinate setup (first time)
2. Advanced calibration (with testing)
3. Quick calibration (fast setup)
4. Set letters and start brute force
5. Test coordinates
6. Quick test word
7. Exit
